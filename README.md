# Link-Prediction-Analysis
Link prediction in complex networks using similarity-based methods, SBM, and Random Forest with evaluation using AUC, AUPR, and Recall@k on real-world datasets.
## ⚙️ Methods Implemented

### 🔹 Similarity-Based Methods

#### Local Methods
- Common Neighbors (CN)
- Jaccard Coefficient
- Adamic-Adar
- Preferential Attachment (PA)
- Resource Allocation (RA)
- Cosine Similarity
- Sorensen Index
- CAR-based methods
- HPI
- LNBCN
- CCLP

#### Global Methods
- Katz Index
- Shortest Path
- SimRank
- LHNG

#### Quasi-Local Methods
- Local Path (LP)
- L3 Index
- CH2-L2
- CH3-L3

---

### 🔹 Probabilistic Model
- Stochastic Block Model (SBM)

---

### 🔹 Learning-Based Model
- Random Forest Classifier

Features used:
- Common Neighbors
- Jaccard Coefficient
- Preferential Attachment

---

## 📊 Datasets

The following real-world datasets were used:

- Adjnoun (word adjacency network)
- Dolphins (social network)
- Football (game network)
- Karate (social network)
- Les Misérables (character co-occurrence)
- Netscience (collaboration network)
- Power Grid (infrastructure network)

---

## 📈 Evaluation Metrics

- AUC (Area Under ROC Curve)
- AUPR (Area Under Precision-Recall Curve)
- Average Precision
- Recall@K
